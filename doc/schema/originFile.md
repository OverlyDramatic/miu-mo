
## originFiles
```js
[
  {
    rootPath: 'D:/abc',
    files: [
      {
        filename: 'music.mp3',
        path: 'D:/abc/defg/music.mp3',
        title: 'music',
        artist: 'abba',
        album: 'baab'
      }
    ]
  }
]
```
## originPath
dirs under the root path that have audio files
```js
[
  {
    rootPath: 'D:/abc',
    paths: [
      'D:/abc',
      'D:/abc/defg'
    ]
  }
]
```
